# Hotstar-Clone-App



## 📝 Description

Hotstar-Clone-App is a meticulously crafted front-end web application designed to replicate the sleek and immersive user interface of the popular Disney+ Hotstar streaming platform. Developed using foundational web technologies—HTML, CSS, and JavaScript—this project serves as an excellent demonstration of modern UI/UX principles. It features a responsive layout, interactive sliders for movie posters, and organized content sections like 'Latest & Trending' or 'Popular Movies.' Whether you are looking for a high-quality starter template for a video streaming project or a comprehensive resource for learning responsive design and DOM manipulation, this clone provides a robust foundation for developers and designers alike.

### Command to RUN 
```
 python -m http.server 5500
```
